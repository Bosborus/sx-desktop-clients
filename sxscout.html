SXScout - File manager

Manage files right from your desktop

SXScout provides the easiest way to manage files in the cluster right from your desktop. It eliminates the need to synchronize files to disk and provides an instant way to download, upload, rename, or delete selected files on any of your remote volumes. It supports all advanced features of SX, including deduplication and client-side encryption.

Features

Fast and lightweight protocol

Never transfer the same data twice. The SX protocol transfers only the differences between the local copy of a file and the data already stored in the cluster. Additionally, it transfers data to/from all SX nodes in parallel to maximize speed.

Encryption

SXScout supports client-side encryption, which not only encrypts the data itself but also the filenames. All communication with the cluster is also by default protected with SSL.

All your volumes in one place

With SXScout you are just one click away from all your volumes and files. Add, delete or rename files and monitor volume usage with a user friendly interface.
